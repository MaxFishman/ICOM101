How to use google webfonts:

  • Pick your font
  • Copy paste this: <link rel="stylesheet" type="text/css" href="https://fonts.googleapis.com/css?family=FONT_NAME_YOU_PICKED"/> in the CSS file of the pen you want to add Google fonts and .....
  • You're done
